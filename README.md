Privacy Policy for Bookmark Manager Extension
=============================================

Overview
--------

The Bookmark Manager extension ("we", "our", or "the extension") is committed to protecting your privacy. This Privacy Policy explains how we handle your data when you use our Chrome extension.

Data Collection and Usage
-------------------------

### What Data We Access

*   **Bookmark Data**: We access your browser bookmarks (titles, URLs, folder structure) to provide the core functionality of organizing and managing your bookmarks
    
*   **Favicons**: We fetch website favicons from Google's favicon service to display visual identifiers alongside your bookmarks
    

### What Data We Do NOT Collect

We do **NOT** collect, store, or transmit:

*   Personal identification information
    
*   Browsing history
    
*   Search queries
    
*   IP addresses
    
*   Device information
    
*   Any analytics or usage data
    
*   Any data to third-party servers
    

Data Storage
------------

### Local Storage

All your bookmark data and extension preferences are stored locally on your device using:

*   Chrome's local storage API
    
*   Chrome's bookmark management API
    

### No External Servers

*   No bookmark data is sent to external servers
    
*   No user data is stored in the cloud
    
*   No personal information is collected or transmitted
    

Permissions Justification
-------------------------

### Bookmarks Permission

The `bookmarks` permission is required to:

*   Read your existing bookmarks for display and organization
    
*   Create new bookmarks when you save web pages
    
*   Update bookmark metadata when you edit them
    
*   Remove bookmarks when you delete them
    
*   Organize bookmarks with tags and categories
    

All bookmark operations occur locally within your browser.

### External Resources

The only external request made is to:

*   `www.google.com/s2/favicons` - to fetch favicons for visual bookmark identification
    

Data Security
-------------

Since all data remains on your local device:

*   Your bookmarks are protected by your device's security
    
*   No data breaches can occur from our servers (we have none)
    
*   You maintain complete control over your data
    

User Control
------------

You have full control over your data:

*   **Access**: View all your bookmarks through the extension interface
    
*   **Modification**: Edit, organize, or delete bookmarks at any time
    
*   **Removal**: Uninstalling the extension removes all locally stored extension data
    
*   **Export**: You can export your bookmarks using Chrome's native bookmark export feature
    

Third-Party Services
--------------------

We do not use any:

*   Analytics services
    
*   Advertising networks
    
*   Tracking technologies
    
*   Third-party data processors
    

Changes to This Privacy Policy
------------------------------

We may update this Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy in the Chrome Web Store listing and updating the "Last Updated" date.

Your Consent
------------

By using our extension, you consent to this Privacy Policy and the local processing of your bookmark data as described herein.


Compliance
----------

This extension complies with:

*   Chrome Web Store Developer Program Policies
    
*   General data protection principles
    
*   Minimal data collection practices
